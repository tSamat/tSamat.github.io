# tSamat — Enterprise AI Lab Portfolio

Public GitHub Pages portfolio for selected enterprise AI, GraphRAG, automation, and platform engineering work.

Site: https://tsamat.github.io/

## Scope

This repository is intentionally lightweight and public-safe. It contains:

- sanitized project descriptions;
- architecture-oriented case study summaries;
- no production data;
- no private documents;
- no credentials;
- no internal infrastructure details.

## Portfolio themes

- Enterprise AI gateways
- GraphRAG and knowledge graphs
- Meeting and report automation
- Kubernetes-based AI operations
- Evaluation and quality loops for RAG systems

## Privacy note

Production systems, client data, internal documents, runtime artifacts, logs, credentials, and environment files are not published. Demo repositories linked from this portfolio should use synthetic fixtures or sanitized subsets only.
